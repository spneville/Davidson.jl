Creating work arrays

Required work arrays

The in-place solver! function can be made essentially allocation free via the passing of pre-allocated work arrays.

Two work vectors have to be supplied:

  • Twork::Vector{T}, where T<:AllowedTypes
  • Rwork::Vector{R}, where R<AllowedFloat

where the types T and R are compatible (see Allowed types)

Dimensions of Twork and Rwork

The required dimensions of the Twork and Rwork vectors are dependent on the size of the matrix being diagonalised, the block size, and the maximum subspace dimension. There exist two functions, Tworksize and Rworksize, that can be used to compute these dimensions.
